# **App Name**: PSIH Threads

## Core Features:

- User Authentication & Profiles: Enable user registration, login (email/Google), and secure management of personal profiles, order history, and wishlists via Firebase.
- Horror Catalog & Filters: Browse PSIH streetwear with filters for category, size, price, and color, presented in a dark, brutalist aesthetic powered by Firestore data.
- Product Immersion & 'Add to Psych Cart': Detailed product pages with multiple images, size selection, and the ability to add items to a dynamic 'Psych Cart', syncing with Firestore when logged in.
- 'Psych Visions' Community Gallery: A community feed where users can upload photos of their PSIH outfits. An AI tool suggests unique horror-themed descriptive captions or stylistic filters for uploaded images, creating a dynamic and immersive social experience.
- 'Psych Filter' Global Visual Toggle: Instantly transform the entire website's visual style into a horror VHS tape aesthetic using heavy CSS and canvas filters, offering an immersive 'Psych' mode.
- Immersive Outfit Builder: Create and save custom streetwear outfits by dragging and dropping clothing items onto a mannequin, with outfits stored securely in user accounts via Firestore.
- Admin Product Management: A secure administrative interface for authorized users to add, edit, and delete products, including image uploads to Firebase Storage, ensuring seamless catalog maintenance.

## Style Guidelines:

- Primary accent: A deep, blood-red (#8B0000) providing stark contrast and an intense focal point against the dark background, used for interactive elements and highlights. HSL(0, 100%, 27%).
- Background: Pure black (#000000) creates a stark, oppressive, and immersive dark-only theme. HSL(0, 0%, 0%).
- Accent: A muted, dark gray (#333333) used for subtle UI elements, secondary text, and borders to provide contrast without overwhelming the primary colors. HSL(0, 0%, 20%).
- Headline font: 'Space Grotesk' (sans-serif) for its computerized, techy feel that complements the glitch and brutalist aesthetic. Body font: 'Inter' (sans-serif) for a modern, objective, and readable appearance, maintaining a minimalistic brutalist typography.
- Utilize 'Lucide' icons for their crisp, minimalist design, maintaining a sharp and consistent visual language across all UI elements, enhancing the brutalist feel.
- Emphasize pure black background and brutalist grid layouts for product displays, fully responsive and mobile-first. Incorporate a dynamic full-screen 'Psych Realm' canvas for an immersive user experience, alongside clear segmentation for product, cart, and account pages.
- Integrate Framer Motion and custom CSS for dynamic, unsettling animations. Heavy use of glitch, VHS, scanline, and film-grain effects on hover states, loading screens, and an animated hero section to encapsulate the brand's horror aesthetic.
